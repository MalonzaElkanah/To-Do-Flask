# to-do App

## Operations
- create a todo list
- update a todo list
- view todo lists
- archive a todo list
- delete a todo list

- add items to a todo list
- delete items from a todo list
- check items from a todo list

## Requirements
- python 3.8
- flask 1.0
- sqlite 
- 

## Run following Commands in shell to run the app.
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
